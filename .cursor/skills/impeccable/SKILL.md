# Impeccable (UI Polish Suite)

This project includes Impeccable sub-skills installed alongside this meta skill.

## Available commands (invoke as separate skills)

| Skill | Use when |
|-------|----------|
| `/impeccable-polish` | Final pass on spacing, alignment, visual consistency |
| `/impeccable-critique` | Review UI issues before shipping |
| `/impeccable-normalize` | Align with design system / Vuetify tokens |
| `/impeccable-animate` | Add subtle motion and transitions |
| `/impeccable-bolder` | Increase visual hierarchy / emphasis |
| `/impeccable-quieter` | Reduce visual noise |
| `/impeccable-audit` | Accessibility and UX audit |
| `/impeccable-delight` | Add delightful micro-interactions |
