# Cursor Skills — KKSK Project

已安装的技能（本地项目级，路径 `.cursor/skills/`）。

## 核心四件套

| 技能 | 调用方式 | 用途 |
|------|----------|------|
| **frontend-design** | `/frontend-design` | 整体视觉、排版、层次感 |
| **vuetify-skilld** | `/vuetify-skilld` | Vuetify 3 组件与主题规范 |
| **impeccable** | `/impeccable` | UI 打磨套件（见下方子技能） |
| **web-design-guidelines** | `/web-design-guidelines` | 可访问性、间距、交互状态 |

## Impeccable 子技能

| 技能 | 调用方式 |
|------|----------|
| polish | `/impeccable-polish` |
| critique | `/impeccable-critique` |
| normalize | `/impeccable-normalize` |
| animate | `/impeccable-animate` |
| bolder | `/impeccable-bolder` |
| quieter | `/impeccable-quieter` |
| audit | `/impeccable-audit` |
| delight | `/impeccable-delight` |

## 使用示例

```
/vuetify-skilld /frontend-design 美化 BlogList 首页，保持 Lothric 深色主题 #2D0425
```

```
/impeccable-polish 优化 LoginForm 间距和卡片层次
```

## 生效方式

安装后执行 **Reload Window**（`Ctrl+Shift+P` → Developer: Reload Window），或在 Cursor **Settings → Rules** 的 Agent Decides 区域查看。
